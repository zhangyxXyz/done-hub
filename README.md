# AI Hub

_AI Hub fork 自 [done-hub](https://github.com/deanxv/done-hub)，done-hub 是基于 [one-hub](https://github.com/MartialBE/one-api) 二次开发的上游项目。_

</div>

## 我们的改动

- 调整项目品牌、Logo、页脚和默认仓库链接。
- 优化访客端首页、价格、关于、聊天等页面的主题表现。
- 新增全局自定义 CSS 配置，便于统一覆盖自定义页面、页脚和站点样式。
- 支持自定义 HTML / iframe 内容跟随系统深浅色主题。
- 优化 fork 环境下的 GitHub Actions、更新检查和发布链接。
- 修复关闭注册后仍显示注册入口的问题。
