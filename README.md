## 我们的改动

- 个性化样式调整，支持全局自定义 CSS、自定义页面、页脚和站点样式统一覆盖。
- 支持自定义 HTML / iframe 内容跟随系统深浅色主题。
- 优化 fork 环境下的 GitHub Actions、更新检查和发布链接。
- 修复关闭注册后仍显示注册入口的问题。
