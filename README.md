# AI Hub

_AI Hub fork 自 [done-hub](https://github.com/deanxv/done-hub)，done-hub 是基于 [one-hub](https://github.com/MartialBE/one-api) 二次开发的上游项目。_

</div>

## 功能摘要

- 个性化样式：全局自定义 CSS、自定义页面、页脚、站点样式统一覆盖。
- 主题适配：自定义 HTML / iframe 内容跟随系统深浅色主题。
- Fork 环境：GitHub Actions、更新检查、发布链接优化。
- 注册入口：关闭注册后的入口隐藏。
